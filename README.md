# [ENG] eParking

A mobile application for managing parking and reserving parking spots.

## Features:
- Find available parking spaces nearby.
- Reserve parking spots.
- Subscription management (Standard, Premium, VIP).
- Google Maps API integration.
- User authentication with Firebase.
- Reservation history.

## Requirements:
- Android 6.0 (API 23) or higher.
- Google account for login.
- Internet connection.

## Installation:
1. Clone the repository:  
   `git clone https://github.com/HubertDragan8/eParking.git`
2. Open the project in Android Studio.
3. Run the app on your device/emulator.

## Technologies:
- Kotlin
- Jetpack Compose
- Google Maps API
- Firebase Auth, Firestore
- Stripe/Google Pay (for payments)

# [PL] eParking

Aplikacja mobilna do zarządzania parkowaniem i rezerwacją miejsc parkingowych.

## Funkcje:
- Wyszukiwanie dostępnych parkingów w pobliżu.
- Rezerwacja miejsc parkingowych.
- Obsługa subskrypcji (Standard, Premium, VIP).
- Integracja z Google Maps API.
- Logowanie i rejestracja użytkowników z Firebase.
- Historia rezerwacji.

## Wymagania:
- Android 6.0 (API 23) lub wyższy.
- Konto Google do logowania.
- Połączenie z Internetem.

## Instalacja:
1. Sklonuj repozytorium:  
   `git clone https://github.com/HubertDragan8/eParking.git`
2. Otwórz projekt w Android Studio.
3. Uruchom aplikację na swoim urządzeniu/emulatorze Android.

## Technologie:
- Kotlin
- Jetpack Compose
- Google Maps API
- Firebase Auth, Firestore
- Stripe/Google Pay (dla płatności)
